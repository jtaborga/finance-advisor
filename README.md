# Finance-Advisor
This repo is for a development challenge called financial advisor.

# React + Foundation
[Foundation](http://foundation.zurb.com/sites/docs/) as [React](https://facebook.github.io/react/) components.

### What do I need? ###
NodeJS
ReactJS

## Build & Development ##
Run `yarn install`

Run `yarn start` to preview
